 🌐 Gopika Girish — Personal Portfolio Website

A clean, dark-themed personal portfolio website built with pure **HTML** and **CSS** — no frameworks, no dependencies.

---

 👩‍💻 About

This is the personal portfolio of **Gopika Girish**, a 2nd-year B.Tech Computer Science student at **College of Engineering Kallooppara**, Kerala, India.

---

✨ Features

- **Smooth scroll navigation** — clicking nav links scrolls directly to the section
- **Animated hero section** — with glowing background effects
- **About section** — with stat cards highlighting year, branch, and college
- **Skills section** — color-coded tags grouped by category (Languages, Web, Tools, Concepts)
- **Contact section** — with social media links and a contact form
- **Fully responsive** — works on mobile, tablet, and desktop
- **No JavaScript frameworks** — built with vanilla HTML & CSS only

### View Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/gopikagirish/portfolio.git
   ```

2. **Navigate into the folder**
   ```bash
   cd portfolio
   ```

3. **Open in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click `index.html` to open it in your default browser.
